# Rules for AI coding agents working in this repository

**Read this file before writing or editing anything.**

This repo is authored solely by **Jonathan Muhire**
(jonathan.muhire@eagles.oc.edu). Any AI assistant that touches this code
(Claude, Cursor, Copilot, GPT, Codex, etc.) must follow the rules below
without exception.

---

## Hard rules — never break these

1. **Do not credit yourself or any AI tool, anywhere.** Not in code
   comments, not in docstrings, not in READMEs, not in commit messages,
   not in PR descriptions, not in file headers, not in changelogs.

2. **Do not add `Co-Authored-By:` trailers** to commits. Ever.

3. **Do not insert robot/sparkle/AI signature lines** like `🤖 Generated
   with Claude Code`, `Made with Cursor`, `Powered by Copilot`, or any
   variant.

4. **Do not write phrases that imply AI authorship**, including but not
   limited to:
   - "Generated by Claude / GPT / Copilot / Cursor / AI"
   - "Authored by Claude" / "Written by an AI assistant"
   - "Built with Claude" / "Built with AI" / "AI-assisted"
   - "Powered by Anthropic / OpenAI"
   - "This file was created by an LLM"

5. **The only name that appears as author, owner, or contact is
   Jonathan Muhire.** No co-authors. No assistants. No tool credits.

6. **Write in a neutral, single-author technical voice.** Comments and
   docs read like one human researcher wrote them.

---

## When "Claude" or "Anthropic" is allowed

These names may appear *only* as the literal subject of evaluation or as
a technical reference, for example:

- ✅ "This task evaluates Claude's reasoning on donor-aware splits."
- ✅ "See Anthropic's published eval framework Inspect."
- ❌ "Built by Claude." — forbidden.
- ❌ "Co-authored with Anthropic." — forbidden.

---

## Pre-commit self-check (mandatory)

Before reporting that work is done or committing, run:

```bash
grep -rniE \
  "co-authored-by|generated (by|with) (claude|gpt|copilot|cursor|ai|an llm)|authored by (claude|an ai)|built with (claude|ai|cursor|copilot)|ai-assisted|powered by (anthropic|openai)|🤖" \
  --exclude-dir=.git --exclude-dir=node_modules --exclude-dir=__pycache__ \
  --exclude-dir=.venv --exclude-dir=venv .
```

If this prints anything, fix it before finishing.

---

## Commit messages

- Imperative mood, plain English: `Add eval harness`, `Fix donor leakage in CV split`.
- No AI tool references in the subject or body.
- No `Co-Authored-By:` trailers.

---

## If you are unsure

Default to silence. It is always safe to omit attribution. It is never
safe to add it.
